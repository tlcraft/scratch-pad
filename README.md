# Scratch Pad

A repo to play with different concepts and ideas in various languages.

## Contents

- [Python Notes](#python-notes)
- [TypeScript Notes](#typescript-notes)

## Python Notes

Change directories into `python` from the root of this project:

```bash
cd python
```

Activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install requirements: 

```bash
pip install -r requirements.txt
```

Save requirements (if needed):

```bash
pip freeze > requirements.txt
```

Run a script:

```bash
python3 src/compose.py
```

Deactivate a virtual environment:

```bash
deactivate
```

## TypeScript Notes

Change directories into the `typescript` directory from the root of this project:

```bash
cd typescript
```

Install dependencies: 

```bash
npm i
```

Run a script from the `package.json` scripts list:

```bash
npm run fetch
```
